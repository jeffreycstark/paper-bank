# Processed Data

This directory contains final, analysis-ready datasets.

## Purpose

Processed data files are:
- Fully cleaned and validated
- Ready for statistical analysis
- Documented with complete metadata
- The authoritative version for all analyses

## Guidelines

- Each file should have a corresponding data dictionary
- Include versioning in filenames if updated
- Document all transformations from raw to processed
- Use efficient formats: `.rds` for R, `.parquet` or `.feather` for Python

## Current Datasets

### Main Analysis Dataset
- **File:** `survey_data.rds` / `survey_data.parquet`
- **Created:** [Date]
- **Script:** `scripts/process_data.R` or `scripts/process_data.py`
- **Variables:** N variables
- **Observations:** N rows
- **Description:** Final analysis dataset with all transformations applied

## Data Processing Pipeline

```
raw/survey_2024.csv
  → scripts/01_clean_data.R
  → interim/01_cleaned_survey.csv
  → scripts/02_recode_variables.R
  → interim/02_recoded_variables.csv
  → scripts/03_finalize_data.R
  → processed/survey_data.rds
```

## Reproducibility

All processed data can be regenerated by running:
```bash
Rscript scripts/build.R
# or
python scripts/build.py
```
