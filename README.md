# EconDev-AuthPref

Research papers on democracy, economic development, and political attitudes in Asia and Latin America.

## Papers

- **meaning-of-democracy-revision** — Electoral winners/losers and democratic conceptions
- **hong-kong-democratic-erosion** — Democratic attitude erosion and sensitivity gradient methodology
- **trust-dynamics** — Institutional trust dynamics across Asian democracies
- **surveys-and-false-positives** — Survey response quality analysis

## Data

This repo consumes harmonized survey data generated by the [`survey-data-prep`](../survey-data-prep/) repository. Paper scripts load data via `_data_config.R` at the project root.

Surveys used: Asian Barometer Survey (ABS), World Values Survey (WVS), Latinobarómetro (LBS).

## Setup

```bash
# R dependencies
Rscript -e "renv::restore()"
```

## Structure

- `papers/` — Manuscript-specific analysis and text
- `data/processed/` — Paper-specific derived datasets (electoral coding, Cambodia subsets)
- `data/external/` — V-Dem scores, COVID economic data
- `outputs/` — Rendered figures and tables
- `scripts/` — Utility scripts (V-Dem download)
